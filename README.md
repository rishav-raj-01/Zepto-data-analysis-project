# Zepto-data-analysis-project
This project focuses on analyzing delivery operations, customer behavior, and sales trends for Zepto, a leading quick-commerce platform. By leveraging SQL, the analysis provides actionable insights into order fulfillment, popular product categories, and regional performance to help optimize business strategies.

Key Objectives :
Operational Efficiency: Analyze delivery lead times and identify bottlenecks in the supply chain.

Customer Insights: Segment customers based on purchasing frequency and average order value (AOV).

Inventory Management: Identify top-selling vs. underperforming products to optimize stock levels.

Revenue Growth: Track monthly and weekly sales growth trends.

Tech Stack :
Data Processing: zepto.csv (Raw Dataset)

Database Management: SQL (MySQL) for structured querying.
